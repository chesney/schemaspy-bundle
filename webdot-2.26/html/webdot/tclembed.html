[previous] [next] WebDot Tutorial: Graphs in Tclets

WebDot can convert a graph into a "Tclet" for the Tcl Plugin. For example, tclembed.dot contains: 
digraph G {
        node [URL="\N.html"]
        edge [URL="\E.html"]
        {a b c} -> d
}
and the source of this page contains (for the Tclet which should be displayed below): 
<webdot src="/cgi-bin/webdot/webdot/tclembed.dot" engine="neato" type="tcl" target="new">

after serving this page through /cgi-bin/webdot, this <webdot...> tag is replaced by: 
<embed src="/cgi-bin/webdot/http://127.0.0.1/cgi-bin/webdot/webdot/tclembed.dot.neato.tcl"
  width="600" height="400">
Graph in Tclet
</embed>
